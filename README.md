# 🛠️ مشروع Backend باستخدام Express

مشروع باك إند بسيط ومنظم باستخدام Express.js  
يدعم التوجيه (Routing) بطريقة منظمة، وسهل التركيب والتوسيع.

---

## 📦 التثبيت

```bash
npm install
# أو
yarn
```
